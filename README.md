## Online modules · 2026

Self-contained interactive modules for the neurobiology teaching block.
Each module is a single HTML file — no installation, no login, no tracking.

---

### Module 1 · Why the brain
**Before week 2 · ~25 minutes**

Opens the neurobiology block by making the idea that *"your mind is physical and it changes"* felt, not just asserted. Built around the London taxi-driver studies (Maguire et al. 2000; Woollett & Maguire 2011), a two-phase learning task, and a private reflection that feeds into the Lecture 1 Mentimeter opener.

→ [Open module](https://BartEllenbroek.github.io/intro-psych-brain/)
